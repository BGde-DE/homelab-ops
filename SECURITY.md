# Security / Disclosure

Dieses Repo enthält ausschließlich anonymisierte Dokumentation und Templates.

## Was niemals im Repo landen wird
- private Keys, VPN-Konfigurationen, PSKs
- vollständige Firewall/Router Exports
- echte Domains/öffentliche IPs/Providerdaten
- Benutzerlisten, Logs, Kamera-/Surveillance-Details

## Umgang mit Beispieldaten
- nur `.example` / Dummy-Werte
- alle Identifikatoren sind Platzhalter (z. B. `service-a.example`, `10.0.0.0/24`)

Wenn mir dennoch versehentlich sensible Daten durchrutschen sollten:
Bitte per Mail/DM melden, ich entferne sie umgehend.
